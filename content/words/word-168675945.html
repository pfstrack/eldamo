ᴹQ. hapalde adv. “*about the same way” (Category: Like, Similar)

⚠️ᴹQ. hapaldë, adv. “*about the same way” (Category: Like, Similar)
ᴺQ. !hapallë “about the same way”

A correlative combination in Demonstrative, Relative, and Correlative Stems (DRC) from 1948 appearing as ᴹQ. hapalde (PE23/111), a combination of ᴹQ. hap- “about the same” and ᴹQ. -l(de) “way”.

Neo-Quenya: For purposes of Neo-Quenya I would update this to ᴺQ. hapallë “about the same way” using the later suffix -llë.

Reference ✧ PE23/111 ✧ “*about the same way”

Elements

hap- “the same, together, alike, closely associated, [of time] approach closely”
-l(de) “like, in the same way/manner”

Ad. voiceless stops became spirants before homorganic spirants; [pf|tθ|kx] > [ff|θθ|xx]
At the same time as aspirates became voiceless spirants, voiceless stops appearing before aspirates of the same series became first affricates and then double spirants (SD/419). The primitive combinations influenced in this way were ✶[ppʰ], ✶[ttʰ] and ✶[kkʰ], which became [ff], [θθ] and [xx], respectively.

Reference ✧ SD/419 ✧ for example: FF < PF < ✶Ad. PPh, ÞÞ < TÞ < ✶Ad. TTh, XX < KX < ✶Ad. KKh

Parma Eldalamberon #12 (PE12)

  • J.R.R. Tolkien, (editors) Christopher Gilson, Carl Hostetter, Patrick Wynne, Arden Smith; 1998

The twelfth issue of Parma Eldalamberon. This issue publishes two documents written in the 1910s: the Qenyaqetsa (PE12/1-28) and the so-called Qenya Lexicon (QL, PE12/29-106); Tolkien himself labeled this document the Qenya Dictionary. The Qenya Lexicon has its own set of entries in this lexicon, so the PE12 entries refer only to the Qenyaqetsa, which is an analysis of Early Qenya phonology.
